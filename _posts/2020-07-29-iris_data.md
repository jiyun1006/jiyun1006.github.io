## 붓꽃 품종 예측   

> ### [STEP1] 데이터 탐색  
> 데이터 유형을 파악하고 알맞게 데이터를 사용한다.   
> 붓꽃 데이터는 ndarray 형태이다.   

#### [코드] 
```
iris_data = load_iris()
print(type(iris_data))

keys = iris_data.keys()
print('붓꽃 데이터 세트의 키들 : ', keys)
```   

#### [결과] 
```
<class 'sklearn.utils.Bunch'>
붓꽃 데이터 세트의 키들 :  dict_keys(['data', 'target', 'target_names', 'DESCR', 'feature_names', 'filename'])
```      
<br/>
<br/>

> ### [STEP2] 데이터 분리   
> 데이터를 학습(train)데이터와 테스트(test)데이터로 나눈다.   
> sklearn.model_selection 모듈의 train_test_split를 이용한다.   

#### [코드]
```
iris_data = load_iris()

X_train, X_test, y_train, y_test = train_test_split(iris_data.data, iris_data.target, test_size = 0.3, random_state=121)
```
*일반적으로 학습 : 테스트 = 7 : 3 비율로 나눈다.*   
<br/>
<br/>
    
> ### [STEP3] 데이터 학습   
> DecisionTreeClassifier를 이용하여 분리한 데이터를 학습시킨다.   

#### [코드]
```
dt_clf = DecisionTreeClassifier()
dt_clf.fit(X_train,y_train)
pred = dt_clf.predict(X_test)
print('예측 정확도 : {:.4f}'.format(accuracy_score(y_test,pred)))
```   

#### [결과]
```
예측 정확도 : 0.9556
```   
<br/>
<br/>

> ### [STEP4] 교차 검증   
> 총 데이터 개수가 적으므로 과적합이 일어날 수 있다. (과적합 : 학습 데이터에만 최적화되어 버리는 현상)
> 학습 데이터를 다시 분할하여 학습시키는 방법

#### [코드]   
```
data = iris_data.data
label = iris_data.target

dt_clf = DecisionTreeClassifier(random_state=156)

scores = cross_val_score(dt_clf, data, label, scoring='accuracy', cv=3)
print('교차 검증별 정확도 : ', np.round(scores, 4))
print('평균 검증 정확도 : ', np.round(np.mean(scores),4))
```   

#### [결과]
```
교차 검증별 정확도 :  [0.98 0.94 0.98]
평균 검증 정확도 :  0.9667
```
